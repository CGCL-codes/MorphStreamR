Detailed Description

The RecordSchema of a particular table.

Properties: fields: an ordered list of column names fieldTypes: an ordered list of data types corresponding to the columns size: physical size (in bytes) of a d_record conforming to this schema

SchemaRecord storage.table.RecordSchema.verify ( List< DataBox >  values) throws SchemaException

Verifies that a list of DataBoxes corresponds to this schema.

A list of DataBoxes corresponds to this schema if the number of DataBoxes in the list equals the number of columns in this schema, and if each DataBox has the same type and size as the columns in this schema.
