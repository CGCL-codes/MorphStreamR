Detailed Description

Implementation of the On-demand Time-decaying Bloom Filter.

It is similar to a time decaying bloom filter, except for the fact that only the counters that are going to be used are updated, instead of all of them periodically.

Creates a new filter with k hash functions accordingly with BloomCalculations#computeBestK(int).

Each bucket will have bucketsPerWord counters, but since this implementation uses a 2D array there's no limitation in the counter value, unlike CountingBloomFilter.
