void scheduler.oplevel.struct.TaskPrecedenceGraph< Context extends OPSchedulerContext >.firstTimeExploreTPG ( Context  context)

During the first explore, we will group operations that can be executed sequentially following Temporal dependencies And all groups constructs a group graph.

And then partition the group graph by cutting logical dependency edges. And then sorting operations in each partition to execute sequentially on each thread, such that they can be batched for execution.
