Detailed Description

A PageAllocation system for an OS paging system.

Provides memory-mapped paging from the OS, an interface to individual pages with the Page objects, an LRU cache for pages, 16GB worth of paging, and virtual page translation.

YOU SHOULD NOT NEED TO CHANGE ANY OF THE CODE IN THIS PACKAGE.
