Detailed Description

Represents a type in a VoltDB.

Note that types in the database don't map 1-1 with types in the Stored Procedure API. Varchars, Chars and Char Arrays in the DB map to Strings in the API.

Object storage.datatype.VoltType.getNullValue ( )

Get a value_list representing whichever null value_list is appropriate for the current VoltType enum.

For example, if this type is TINYINT, this will return a java byte with value_list -128, which is the constant NULL_TINYINT in VoltDB.

A new final instance with value_list equal to null for a given type.
