Detailed Description

A plug-in replacement for JDK1.5 java.util.concurrent.ConcurrentHashMap.

This version is based on org.cliffc.high_scale_lib.NonBlockingHashMap. This solution should be completely compatible, including the serialized forms and all multi-threaded ordering guarantees.

Parameters
<K>the type of keys maintained by this map
<V>the type of mapped values
