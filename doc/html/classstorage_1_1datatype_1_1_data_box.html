Detailed Description

Abstract DataBox for all database primitives Currently supported: integers, booleans, floats, fixed-length strings.

DataBoxes are also comparable allowing comparisons or sorting.

Provides default functionality for all DataBox subclasses by assuming that the contained value_list is not of the type specified.
