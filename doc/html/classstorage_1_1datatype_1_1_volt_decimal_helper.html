Detailed Description

A class for serializing and deserializing Volt's 16-byte fixed precision and scale decimal format.

The decimal's are converted to/from Java's BigDecimal class. BigDecimal stores values as an unscaled (unscaled means no trailing 0s) fixed point BigInteger and a separate scale value_list. An exception (either RuntimeException or IOException) if a BigDecimal with a scale > 12 or precision greater then 38 is used. BigDecimal.setScale(int) can be used to reduce the scale of a value_list before serialization.
